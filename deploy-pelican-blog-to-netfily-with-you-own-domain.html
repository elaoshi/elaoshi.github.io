Deploy Pelican website blog to Netfily

Posted on Wed 08 April 2020 in Tech

First we have a pelican website. Then I deploy my code to bitbucket.

In my repo, need to create a requirements.txt file. This is important as Netlify will look for this file to determine dependencies. In the file:

    Markdown==2.6.11
    pelican==4.2.0
    requests==2.18.4
    s3cmd==2.0.1
    six==1.14.0
    typogrify==2.0.7
    pytest==4.6.9

After then, I connect netlify with bitbucket, netlify ask me to choose my own repo to pull.

Then when I config the site in netlify: 

    Build command:   pelican content -s publishconf.py
    Publish directory:   output

After that, I choose master branch to build.

Finally , Click the Deploy button to build and deploy to netlify.

Point to my own domain

In setting : Deploy -> Domain management

I add a domain alias and point my domain dns to netlify .

Then netlify will create a DNS panel for me to manage my domain. So sweet!

After dns name server pointed to netlify, we can create HTTPS certificate by Let's Encrypt for free!

Till now, my blog is already setup! Sweet!

Update content

In my docker instance, I can generate new content by running : 

    pelican content -s publishconf.py

Then add a build "Build hooks" in "Build & deploy" , netfily give me a url.

I set this url in bitbucket webhook, when each time content pushed to bitbucket then bitbucket will trigger a build in netfily.

Till now, when I deploy content to bitbucket and netlify will auto deploy it to my blog! Sweet!
