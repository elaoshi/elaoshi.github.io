Integrated Pelican website blog with Circle CI

Posted on Thu 09 April 2020 in Tech

My blog already integrated with Cricle CI

Good for me to testing both in Cirecle CI and Netlify.

The structure is following :

v1:

v2:

My circle config yml v1 is : 

version: 1
jobs:
  build:
    docker:
      - image: circleci/python:3.7
    dependencies:
      pre:
        - pip install --upgrade pip
    steps:
        - checkout
        - run:
            command: |
                ls /
                python3 -m venv venv
                . venv/bin/activate
                pip install -r requirements.txt
        - run:
                name: Running tests
                command: |
                . venv/bin/activate
                pytest
        - store_artifacts:
            path: test-reports/
            destination: python_app
